🌌 Overview

OSN (Global Intelligence) is a unified security analysis framework designed to process global threat vectors and generate high-fidelity, command-center-style intelligence reports. The system integrates real-time data processing, automated analysis, and futuristic visualization into a single, high-performance execution flow.

🚀 Key Features

📡 Live Global Uplink: Automated real-time extraction of security events from international RSS nodes (Reuters, BBC).

🖥️ NASA-Style Command Center UI: A futuristic, dark-mode terminal report featuring Glassmorphism, neon-glow threat indicators, a live news ticker, and real-time scanline effects.

🧠 Automated Threat Assessment: Algorithmic classification of global incidents (DEFCON / ALPHA-RED / BRAVO-YELLOW) based on keyword telemetry.

⚙️ Zero-Intervention CI/CD: Fully integrated with GitHub Actions for automated hourly scans and immediate deployment to GitHub Pages.

🛠️ System Architecture

The core operation follows a strict three-stage execution pipeline:

Intelligence Fetching (Uplink): Connects to global RSS feeds, extracting raw data and calculating time-to-event telemetry.

Analysis Engine (Processing): Evaluates threat vectors (Critical, High, Medium, Low) and calculates system-wide alert statuses and node activity.

Visualization Layer (Deployment): Compiles the intelligence into a single-file index.html payload and pushes it to the live server.

🖥️ Installation & Usage

Local Terminal Access

To run the analysis and boot the command center locally:

# 1. Clone the repository
git clone [https://github.com/osk0020/global-security-data-analysis.git](https://github.com/osk0020/global-security-data-analysis.git)

# 2. Install dependencies
pip install requests beautifulsoup4 feedparser

# 3. Initialize the uplink
python analysis.py


System will automatically open the generated index.html report in your default browser.
